# &lt;call-display&gt;

## Methods

### `setCallType()`

Sets the type of call.

#### Arguments

- `video`
- `audio`

### `setCaller()`

Sets the callers username.

## Custom Events

### `acceptCall`

Triggered when the user accepts the call.

### `denyCall`

Triggers when the user denies the call.
